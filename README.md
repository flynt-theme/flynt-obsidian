# Flynt for Obsidian

Warm tones. Zero visual noise.

![Flynt - light and dark](screenshot.png)

## About

Flynt is a theme built around a simple idea: most themes make the *structure* of your notes louder than the *content*. Flynt inverts that. Syntax and formatting recede; what you wrote takes center stage.

- Headings descend through a warm text scale - no color, just contrast
- Amber for links and list markers - the connective tissue of your notes
- Moss for inline code and fenced blocks
- Bold and italic sit just above and below body text, not fighting for attention
- Full dark and light support

## Installation

Search for **Flynt** in Settings - Appearance - Themes.

## Manual Installation

1. Download `theme.css` from this repository
2. Copy it to `<vault>/.obsidian/themes/Flynt/theme.css`
3. Select Flynt in Settings - Appearance - Themes
